# Ideakeeper

A product asset library and automation toolset.

## Features
- Asset management and extraction.
- Support for Douyin, Taobao, and other platforms.
- Frontend interface for browsing and editing product libraries.

## Setup
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Configure environment variables in `.env`.
4. Run the application: `python main.py`.
